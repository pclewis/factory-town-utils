How to Use

  1. Back up your save file before you do anything
  2. Place an agent sensor on the edge of the area you want to copy
  3. Change the inventory filter to the very last option, "FilterItemCategories"
  4. Set the offset to the opposite corner of the area you want to copy (you can place a block to click if needed and remove it after)
  5. Place any kind of logic block where you want to copy the area
  6. Click on the initial agent sensor, and then right click on the new block to create a logic link
  7. Save your game
  8. Upload your save file from C:\Program Files (x86)\Steam\userdata\[your account id]\860890\remote
  9. Click Process
  10. Click download link, save file in C:\Users\[your name]\AppData\LocalLow\82 Apps\Factory Town\Saved Games

You can copy several areas to several other places at once, just place more agent sensors with the correct filter and link them to as many targets as you want.
